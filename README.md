# عداد الركعات - Prayer Rak'ah Counter

تطبيق Android لعد ركعات الصلاة تلقائياً باستخدام حساس التقارب.

---

## كيفية عمل التطبيق

1. ضع الهاتف أمامك على الأرض عند الصلاة
2. أدخل عدد الركعات المطلوبة واضغط **ابدأ**
3. عند السجود، يقترب رأسك من الهاتف → يتحول الشاشة للأسود + يُحسب سجود
4. كل سجودتين = ركعة واحدة
5. كل ركعتين تظهر رسالة **"تشهد وسلم"** لمدة 5 ثوانٍ
6. عند اكتمال الهدف تظهر رسالة **"تقبل الله صالح الأعمال"**

---

## متطلبات البناء

- Android Studio (Hedgehog أو أحدث)
- JDK 8 أو أعلى
- Android SDK 34
- جهاز Android يحتوي على حساس تقارب (Proximity Sensor)

---

## طريقة البناء والتثبيت

### 1. افتح المشروع
افتح Android Studio واختر **Open** ثم اختر مجلد `PrayerCounter`

### 2. انتظر مزامنة Gradle
سيقوم Android Studio بتحميل التبعيات تلقائياً

### 3. اضغط Run أو Shift+F10
اختر جهازك المتصل أو المحاكي

---

## بناء APK للتثبيت المباشر

من Android Studio:
```
Build → Build Bundle(s) / APK(s) → Build APK(s)
```
ستجد الملف في:
```
app/build/outputs/apk/debug/app-debug.apk
```

## ملاحظات تقنية

- **حساس التقارب**: يُكشف عند مسافة أقل من 5 سم
- **Debounce**: 1.5 ثانية بين كل سجودتين لتجنب التكرار
- **الشاشة تبقى مضاءة** طوال فترة الاستخدام (FLAG_KEEP_SCREEN_ON)
- يدعم اللغة العربية بشكل كامل

## الشاشات

| الشاشة | الوصف |
|--------|-------|
| شاشة الإعداد | إدخال عدد الركعات + زر ابدأ |
| شاشة العد | عرض الركعات + السجدات |
| السجود | تحول الشاشة للأسود |
| التشهد | رسالة حمراء كل ركعتين |
| الاكتمال | رسالة "تقبل الله صالح الأعمال" |
